# Portafolio Personal - Flutter App

Este proyecto es una aplicación móvil multiplataforma desarrollada en Flutter y Dart para la materia Programación III (Universidad de Margarita - UNIMAR). Consiste en un portafolio interactivo estructurado de forma modular y personalizada.

# Inspiración y Concepto Visual
El diseño estético de la aplicación está inspirado en el anime Black Clover. Utiliza una interfaz en modo oscuro profundo con acentos en tonos negros/rojos y tipografías de estilo medieval/gótico para emular la apariencia del aura del grimorio de cinco hojas.

# Requisitos Obligatorios Cumplidos
- Estructura de Carpetas: Segmentación limpia del proyecto organizando vistas en "screens", componentes reutilizables en "widgets" y estilos globales en "themes".
- Pantalla Principal (Home Screen): Incluye presentación del estudiante, sección de historia (¿Quién soy?) y una lista de intereses (Mis Hobbies).
- Pantalla de Contacto y Firma: Vista secundaria accesible desde la barra de acciones superior (AppBar) a través de navegación nativa por rutas.
- Componentes Reutilizables: Implementación de un pie de página (CustomFooter) modularizado para optimizar la legibilidad del código.
- Control de Versiones: Historial de desarrollo registrado paso a paso mediante confirmaciones locales y remotas en GitHub.

# Dependencias Utilizadas
Para lograr la personalización estética del software se integraron las siguientes librerías del ecosistema oficial de Dart:

1. "google_fonts": Implementada para la gestión dinámica de tipografías premium directas desde los servidores de Google sin sobrecargar el peso del instalador.
   - Fuentes utilizadas: principalmente "unifrakturCook" (estilo gótico/medieval para títulos en un intento de emular el logotipo oficial de la serie) y "Poppins" (fuente geométrica de alta legibilidad para textos de cuerpo).
